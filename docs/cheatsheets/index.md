# Cheatsheets

Quick-reference sheets for CDE Analyzer commands, concepts, and workflows.
Each cheatsheet is a standalone 2-page HTML document optimized for landscape printing.

**Tip:** Open in a browser and print to PDF (landscape orientation) for best results.

---

## Available Cheatsheets

### [Core Concepts & Data Model](concepts.html)

CDE structure, field paths, pattern TSV format, file types, k-mer mining algorithm,
subsumption and coalescing, temporal patterns, and three-layer architecture.

### [Command Reference](commands.html)

All 26 CLI commands organized by category: mining, instrument extraction,
discovery & stripping, pattern utilities, reporting, data extraction, and workflow
orchestration. Includes the standardized short options reference table.

### [Pipeline & Workflow](pipeline.html)

Three-phase pipeline overview (instruments, phrases, branching strip), curation
workflow cycle, workflow engine YAML format, configuration priority, and
copy-paste command recipes for common tasks.

### [TSV Pattern Editor](tsv-editor.html)

Browser-based curation tool cheatsheet: editor interface mockup with all visual
cues, the 4 curation decisions (keep/remove/modify/substitute), keyboard
shortcuts, merge groups with propagation, column filters, and pipeline
integration.
